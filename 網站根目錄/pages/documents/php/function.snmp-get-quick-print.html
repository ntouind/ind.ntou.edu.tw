snmp_get_quick_print

snmp_get_quick_print -- Fetch the current value of the UCD library's quick_print setting

Description

boolean snmp_get_quick_print(void );

Returns the current value stored in the UCD Library for quick_print. quick_print is off by default. 

 
$quickprint = snmp_get_quick_print();

Above function call would return 0 (false) if quick_print is on, and 1 (true) if quick_print is on.

snmp_get_quick_print() is only available when using the UCD SNMP library. This function is not applicable to Windows NT.

See: snmp_set_quick_print() for a description of what quick_print actually does.
