PDF_show_xy

PDF_show_xy -- Output text at given position

Description

void pdf_show_xy(int pdf document, string text, double x-koor, double y-koor);

The PDF_show_xy() function outputs the string text at position (x-koor, y-koor).

See also PDF_show().
