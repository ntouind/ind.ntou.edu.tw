PDF_open_gif

PDF_open_gif -- Opens a GIF image

Description

int pdf_open_gifg(int pdf document, string file name);

The PDF_open_gif() function opens an image stored in the file with the name file name. The format of the image has to be jpeg.

See also PDF_close_image(), PDF_open_jpeg(), PDF_open_memory_image(), PDF_execute_image(), PDF_place_image(), PDF_put_image().
