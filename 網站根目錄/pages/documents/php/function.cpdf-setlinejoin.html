cpdf_setlinejoin

cpdf_setlinejoin -- Sets linejoin parameter

Description

void cpdf_setlinejoin(int pdf document, long value);

The cpdf_setlinejoin() function set the linejoin parameter between a value of 0 and 2. 0 = miter, 1 = round, 2 = bevel.
