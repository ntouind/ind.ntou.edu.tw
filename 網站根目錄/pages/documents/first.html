中文 locale 問題

請在命令列下以下指令 
ind:~> locale
LANG=
LC_COLLATE="en_US.ISO8859-1"
LC_CTYPE="en_US.ISO8859-1"
LC_MONETARY="en_US.ISO8859-1"
LC_NUMERIC="en_US.ISO8859-1"
LC_TIME="en_US.ISO8859-1"
LC_MESSAGES="en_US.ISO8859-1"
LC_ALL=en_US.ISO8859-1
ind:~>
若非以上所示, 請更改 locale 設定為 en_US.ISO8859-1 
For tcsh, csh:
	setenv LC_ALL en_US.ISO8859-1

For sh, bash:
	set LC_ALL=en_US.ISO8859-1
	export LC_ALL

可將其加入 /etc/csh.cshrc (for tcsh & csh) 或 /etc/exports (for sh & bash)
裡, 以便在 users login 時讀取. 

	其他 locale type 可參照 /usr/lib/nls/loc 裡之說明
